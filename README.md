# Анализ финансовой отчётности крупных компаний (2009–2023)

## Описание проекта
Исследование финансовых показателей крупнейших компаний мира на основе открытых данных:
- Анализ выручки, чистой прибыли и рентабельности
- Расчёт ROA (рентабельности активов)
- Выявление трендов за 14 лет (2009–2023)
- Визуализация результатов

## Стек технологий
- **Python** (pandas, matplotlib) — обработка и визуализация данных
- **SQL / PostgreSQL** — аналитические запросы
- **Excel / Power BI** — дашборд с итогами

## Датасет
[Financial Statements of Major Companies (2009–2023)](https://www.kaggle.com/datasets/rish59/financial-statements-of-major-companies2009-2023) — Kaggle (открытые данные)

## Структура проекта
```
├── analysis.py                      # Основной Python-скрипт анализа
├── queries.sql                      # SQL-запросы для PostgreSQL
├── financial_analysis_dashboard.png # Визуализация результатов
├── top10_by_revenue.csv             # Топ-10 по выручке
├── top10_by_roa.csv                 # Топ-10 по ROA
├── yearly_trends.csv                # Динамика по годам
└── README.md
```

## Как запустить
```bash
# 1. Установите зависимости
pip install pandas matplotlib

# 2. Скачайте датасет с Kaggle и положите в папку проекта как financial_statements.csv

# 3. Запустите анализ
python analysis.py
```

## Ключевые результаты
- Рассчитана рентабельность (Net Margin) и ROA для всех компаний
- Выявлены топ-10 компаний по выручке и эффективности активов
- Построена динамика средней рентабельности рынка 2009–2023
- Определены компании с устойчивым ростом выручки 10+ лет подряд
